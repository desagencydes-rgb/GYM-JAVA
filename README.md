# Gym Management App

## How to Run

### Option 1: Using an IDE (Recommended)
1.  Open this folder (`C:\Projects\gym`) in **IntelliJ IDEA**, **Eclipse**, or **VS Code**.
2.  The IDE should automatically detect the `pom.xml` file and download the necessary dependencies (JavaFX, SQLite, etc.).
3.  Locate the main class:
    `src/main/java/com/gym/app/Main.java`
4.  Right-click the file and select **Run 'Main'**.

### Option 2: Using Maven (Command Line)
If you have Maven installed and added to your PATH:
1.  Open your terminal/command prompt.
2.  Navigate to the project directory:
    ```powershell
    cd C:\Projects\gym
    ```
3.  Run the application using the JavaFX plugin:
    ```powershell
    mvn javafx:run
    ```
